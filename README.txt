# Frontend UI (login + CFDI)
1) Abre `index.html` (Vercel o local).
2) Ve a la pestaña **Sesión**:
   - Pon **API Base** (ej. `http://localhost:4000/api`).
   - Clic **Crear usuario demo** y luego **Iniciar sesión** (token queda guardado).
3) En **Facturación**, sube **XML** (y PDF) → se crea la factura en backend y se muestra el preview + se agrega al listado local.
